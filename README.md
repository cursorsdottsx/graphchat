# Reanvue

The Developer's Connection

## Development

### Web

```bash
# Install dependencies
$ yarn

# Serve
$ yarn dev:web

# Build for production and launch server
$ yarn build:web
$ yarn start:web
```

### Server

```bash
# Install dependencies
$ yarn

# Serve
$ yarn dev:server

# Build for production and launch server
$ yarn build:server
$ yarn start:server
```
